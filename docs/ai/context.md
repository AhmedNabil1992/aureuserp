# AI Context

This is a Laravel web application built on the TALL stack, with Filament v3 as a core UI framework.

Filament Panels are used to build the ENTIRE application, including the customer-facing App Panel and the Admin Panel.

Filament is not used as a backend-only dashboard. UI/UX decisions must prioritize end-user experience, clarity, and performance.

Prefer Filament custom themes, panel-aware navigation, and frontend-friendly layouts over generic admin-only patterns.
