# AI Coding Standards

Follow professional Laravel and PHP standards across all changes:

- Comply with PSR-12 formatting and naming conventions.
- Use strict typing in PHP files whenever applicable.
- Prefer modern PHP 8+ language features where they improve readability and safety.
- Use clear, descriptive class, method, and variable names.

Filament-first development rules:

- Prefer Filament fluent APIs, components, forms, tables, actions, and schemas over custom HTML or Blade implementations whenever possible.
- Reuse existing panel components and patterns before introducing new abstractions.
- Keep UI definitions consistent with existing panel conventions.
